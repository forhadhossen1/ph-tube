Some Discussion

Var

The var statement declares a variable. Variables are containers for storing information.

Hoisting: Variable declarations using var are hoisted to the top of their containing function or global scope, but not their initializations.

Re-Declare: You can re-declare a variable using var within the same scope

Let

Block-scoped: The keyword let in Javascript is used to declare a variable that is block scoped in JavaScript.

Hoisting: Variable declarations using let are hoisted to the top of their block, but not their initialized.

Re-Declare: You can not re-declare a variable using let within the same scope

Const

Block-scoped: The const declaration declares block-scoped local variables.

Hoisting: Variable declarations using const are hoisted to the top of their block, but not initialized.

Re-Declare: You can not re-declare a variable using let within the same scope

API

REST API stands for Representational State Transfer Application Programming Interface. It is a specific type of web-based API that follows a set of architectural principles and constraints for designing networked applications. REST APIs use HTTP requests to perform various operations on resources, which can be data objects, services, or entities, typically represented in a structured format like JSON or XML.
